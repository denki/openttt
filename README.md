openttt
=======
Software for managing table tennis tournaments. Written in java.

Requirements
------------
* java-jdk >= 1.7
* apache-ant (installing, building the JAR)

Download
--------
http://wwwpub.zih.tu-dresden.de/~s7480817/openttt.jar

Compile
-------
$ make

Run
---
$ make test

Build JAR file
--------------
$ make jar

Install
-------
\# make install

Uninstall
---------
\# make uninstall
